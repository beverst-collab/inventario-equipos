# Sistema de Inventario de Equipos

Aplicación web desarrollada en Django para la gestión de equipos de cómputo con control de usuarios y roles.

##  Funcionalidades

- CRUD de equipos (Crear, Leer, Actualizar, Eliminar)
- Sistema de autenticación (Login / Logout)
- Control de roles (Administrador / Técnico)
- Filtros por tipo y estado
- Búsqueda avanzada
- Interfaz moderna con Bootstrap

##  Tecnologías

- Python 3
- Django
- SQLite
- HTML / CSS / Bootstrap

##  Roles

### Administrador
- Crear equipos
- Editar equipos
- Eliminar equipos

### Técnico
- Visualizar equipos
- Filtrar información

python manage.py runserver
